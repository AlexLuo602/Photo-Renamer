This project changes the names of the files in your specificed folder into the date it was created 
Format: 2020-12-13 17꞉39꞉32 (example) (this will also be the name of your file)

Instructions:
1. Open main.exe file through dist --> main.exe
2. Past the path of the folder you want to rename and press enter
3. If you are finished, press any key
4. Otherwise, press yes or y to rename a different folder
